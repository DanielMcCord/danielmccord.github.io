Target Shooter

Download for Android | GitHub

Year: 2019

A target shooting game in Solar2D.

This was a school assignment, with more math involved than most of my other work. I did some trigonometry to make the turret turn based on where the screen is tapped. I used relative coordnate systems for collision detection, random numbers for creating targets, and some more math to work around the fact that the assignment required me to implement the bullets using transitions instead of regular motion.
